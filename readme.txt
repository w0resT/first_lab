Задание 24. Дан список вещественных чисел. Все его элементы, не равные нулю, переписать, сохраняя порядок, в начало списка, а нулевые элементы в конец списка. Новый список заводить нельзя.
Для корректного запуска требуется файл 'file.txt', где храняться вещественные числа. Путь файла можно поменять в строке 393 в функции 'file_processing'.
Пример заполнения файла 'file.txt': "0.1, 0.0, 1.11, 6.9, 0.0, 4.22, 0.0, 0.313"
Запуск программы осуществлять при помощи консоли или добавив перед 'return EXIT_SUCCESS;' в функции main строчку 'system("pause");' как обычное приложение.
Тестировалось на Windows 10.
